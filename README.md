# Servidor Backend - Exercício de API com GET e POST

## 📝 Projeto
 → Este projeto foi desenvolvido como uma atividade prática no SENAI para criar um servidor simples em Python. O objetivo foi aprender o funcionamento de requisições HTTP, utilizando os métodos GET e POST e realizando testes de API com o Postman.


## 🖥️ Tecnologias Utilizadas
- Python
- Postman
- Visual Studio Code (VS Code)
- Git e GitHub
